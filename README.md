## Launching & Operating Commands
```JS
1. run main.py
2. Login (enter username and password)
3. User-Interfaces	
	User UI:
	Press 0-3 then Return depending on your desired action:
	    0. Exit
	    1. Browse
	    2. Search
	    3. Track Order
	
	Owner UI:
	Press 0-3 then Return depending on your desired action:
	    0. Exit
	    1. Add Book
	    2. Remove Book
	    3. View Basket
	    4. Print Report
```
