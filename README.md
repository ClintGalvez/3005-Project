	- MEMORY LEAK TEST: valgrind --leak-check=yes ./bookstore
2. Login (enter username and password)
3. User-Interfaces

	Press 0-3 then Return depending on your desired action:
	    0. Exit
	    1. Browse
	    2. Search
	    3. Track Order
	
	Owner UI:
	Press 0-3 then Return depending on your desired action:
	    0. Exit
	    1. Add Book
	    2. Remove Book
	    3. Print Report
